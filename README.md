# Stock Price Prediction using Machine Learning

This project predicts future stock prices based on historical data using machine learning techniques.

## Project Description
A machine learning model is trained on historical stock price data to predict future price trends. The system uses linear regression for prediction and visualizes the results using graphs.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Features
- Historical stock price analysis
- Future price prediction
- Graphical visualization of predictions

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run the program: python stock_prediction.py
